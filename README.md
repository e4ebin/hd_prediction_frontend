# Documentation

## Run Locally

Clone the project

```bash
  git clone https://github.com/e4ebin/hd_prediction_frontend
```

Go to the project directory

```bash
  cd hd_prediction_frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
